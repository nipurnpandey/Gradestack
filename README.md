🎓 GradeStack

GradeStack is a centralized academic platform built for students of CSJMU-affiliated universities, specifically for BCA and BBA programs.

It provides structured study materials, previous year question papers, and a smart progress tracking system — all in one place to help students study efficiently and stay organized.

🚀 Features
📚 Comprehensive Notes
Subject-wise notes for BCA & BBA
Semester-wise structured content
📄 Previous Year Question Papers
Easy access to past exam papers
Helps in exam preparation and pattern understanding
📊 Progress Tracker
Monitor your study progress
Stay consistent with your learning goals
🔐 User Authentication
Secure login and signup system
Personalized dashboard experience
🌐 All-in-One Study Platform
No need to switch between multiple sources
Everything available at one place
🎯 Purpose

GradeStack aims to:

Simplify access to quality study material
Help students stay organized and consistent
Provide a structured academic ecosystem
Improve exam preparation efficiency
